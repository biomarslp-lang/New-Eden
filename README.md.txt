# New-Eden

Ein einfaches Weltraum-Browserspiel, 

## Aktuelle Funktionen
* Ressourcen-Generierung (Metall)
* Ausbau der Metallmine
* Kosten-Skalierung pro Stufe

## Techniken
* HTML5
* CSS3 (Space-Design)
* JavaScript (Spiel-Logik)

## Geplante Features
* Kristall-Ressourcen-System
* Schiffswerft und Flottenbau
* Speichersystem (LocalStorage)
